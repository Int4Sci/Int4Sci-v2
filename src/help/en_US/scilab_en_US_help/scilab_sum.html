Calling Sequence

y = x + z

Parameters

y

interval matrix or vector

x,z

real/interval matrix or vector

Description

  • This operation allows to add an interval matrix or vector with a same size interval or real matrix/vector. Such an addition is defined ([Hansen03], [Moore66]) by an addition of the inferior bounds and an other one of the superior bounds:

    y = x + z with x = |xi,xs| and z = |zi,zs| gives y = |xi+zi,xs+zs|.

    y = x + z with x real and z = |zi,zs| gives y = |x+zi,x+zs|.

    Such an addition is here extended elementwise for same size interval matrices and vectors.

    A Scilab characteristic is the application of basic arithmetic operations between matrices/vectors and single variables (by making the operation between each component of the matrix/vector input and the single varaible). The addition for intervals or for intervals/reals available on Int4Sci can also deal with such operations.

    All these operations are executed by the addition functions for intervals given by the library BIAS [Rump99].

Examples

#(2,3) + 3;
9 + #(-5,6);
#(0,2) + #(3,4);

A = [1,2,3;3,4,5];
B = [2,3,4;4,5,6];
C = interval(A,B);
D = interval(-A,B);

#(0,2) + C;
C + 4;

C + D;
C + D.inf;

Calling Sequence

y = x - z

Parameters

y

interval matrix or vector

x,z

real/interval matrix or vector

Description

  • This operation allows to substract an interval matrix or vector to a same size interval or real matrix/vector. Such a substraction is defined ([Hansen03], [Moore66]) as the addition with the opposite interval :

    y = x - z with x = |xi,xs| and z = |zi,zs| is equivalent to y = x + (-z) with - z such as -z = |-zs,-zi|.

    Substraction can also be done between intervals and reals :

    y = x - z with x = |xi,xs| and z real gives y = |xi-z,xs-z|.

    y = x - z with x real and z = |zi,zs| gives y = |x-zi,x-zs|.

    Such a substraction is here extended elementwise for same size interval matrices and vectors.

    The substraction for intervals or for intervals/reals available on Int4Sci can also deal with operations including a single variable and matrix/vector variable.

    All these operations are executed by the addition functions given by the library BIAS [Rump99].

Examples

#(2,3) - 3;
9 - #(-5,6);
#(0,2) - #(3,4);

A = [1,2,3;3,4,5];
B = [2,3,4;4,5,6];
C = interval(A,B);
D = interval(-A,B);

#(0,2) - C;
C - 4;

C - D;
C - D.inf;

Calling Sequence

y = x * z
y = x .* z
y = x .*. z

Parameters

y

interval matrix or vector

x,z

real/interval matrix or vector

Description

  • A basic multiplication is defined for two single intervals [Moore66], [Hansen03] :

    y = x * z with x = |xi,xs| and z = |zi,zs| gives y = |min(B), max(B)| with B the set B = { xi*zi, xi*zs, xs*zi, xs*zs}.

    An interval can also be multiplicated by a real :

    y = x * z with x = |xi,xs| and z real gives y = |xi*z,xs*z|, which is comutative.

    These operations are the base of the three multiplications implemented on this interface :

    - * is the matrix/matrix multiplication and the multiplication defined above with single elements. It's then possible to multiplicate two interval matrices, A by B for example ( A with a number of columns equal to the number of rows of B ), following the same rules than a real matrices multiplication ( scalar product of each row of A by each column of B) with interval arithmetic operations (addition and multiplication defined above). * is available too for interval matrix/vector multiplication and real matrix/interval matrix multiplication (based on the real/interval multiplication defined just above). Finally, a matrix/vector of intervals can be multiplicated elementwise by a single interval with * (and so with intervals and reals).

    - .* is the elementwise multiplication for two same size interval matrices or vectors. It's available too for an interval matrix/vector and a same size real matrix/vector. .* is equivalent to * for matrix/vector multiplication with single elements (as for intervals than for reals) and for multiplication defined just above.

    - .*. is the Kroneker multiplication (see kron function for detailed definition) that is here extended for intervals and reals.

    All these operations are executed by the multiplication functions given by the library BIAS [Rump99].

Examples

#(2,3) * 3;
9 * #(-5,6);
#(0,2) * #(3,4);

A = [1,2,3;3,4,5];
B = [2,3,4;4,5,6];
C = interval(A,B);
D = interval(-A,B);

#(0,2) * C;
C * 4;

C * D; // matrix matrix mutiplication
C .* D; // elementwise multiplication
C .*. D; // Kroneker multiplication

C * D.inf; // matrix matrix mutiplication for intervals and reals
C .* D.inf; // elementwise multiplication between reals and intervals
C .*. D.inf; // Kroneker multiplication between reals and intervals

Description

  • A basic division is defined for two single intervals [Moore66], [Hansen03] :

    y = x / z with x = |xi,xs| and z = |zi,zs| gives y = x * 1/z with 1/z such as 1/z = | 1/zs, 1/zi|. If zi is zero, 1/zi is assumed to beÂ the infinite value +inf. In a same way, if zs is zero, 1/zs is -inf.

    An interval can also be divided by a real :

    y = x / z with x = |xi,xs| and z real gives y = |xi/z,xs/z|.

    Finaly, a real can be divided by an interval :

    y = z / x with x = |xi,xs| and z real gives y = |z/xs,z/xi|.

    Such operations for single intervals and reals have mathematical problems when the divider contains or is zero. Different conventions exists to deal with such cases. We have choosen here those :

    - The divsion by an interval containing zero will return the set of real numbers represented by |-inf,inf| (Hansen convention [Hansen03]).

    - The division of an interval by zero or by the degenerate interval |0,0| doesn't exist (classical mathematic convention) and then returns the empty set |Nan,Nan|.

    - A convention for division by an interval including zero exists too : it gives an union of the division by the positive part and the division by the negative part. The function listdiv has been implemented to allow such result for single intervals or a real divided by an interval. For example, listdiv(1,#(-a,a)) will return the list |-inf,-1/a| , |1/a,inf|.

    With the two first conventions, we have implemented in this interface operations for interval matrices or vectors :

    - / : For real square matrices A and B, the operation A/B consists to multiply (matrix/matrix) A with the inverse matrix of B. For intervals, the inversion of an interval matrix doesn't exist. So / is used to divide the components of an interval matrix or vector by a single interval and divide a single interval by the components of an interval matrix or vector. The result is stored in an interval matrix or vector. The same operations are allowed between reals and intervals. We can notice that, as a real square matrix can be inversed, the operation A/B, with A a square interval matrix and B a same size real matrix, is authorized and is equivalent to A * B^(-1).

    - ./ : For two same size interval matrices or vectors A and B, A./B is an elementwise division. As allowed by classical Scilab use, this operator can be used with a single interval and an interval matrix/vector, but it's here equivalent to the use of the / operator. Here again, we can mix with this operator real and interval values.

    - \ : This operator is comonly used for linear system resolution. In this version, applied to intervals (or intervals and reals between them), this operator is only the "inverse" of / (meaning A/B is equivalent to B\A) and keeps then the same uses and restrictions than /.

    - .\ : The operation A.\B for intervals or intervals and reals between them is equivalent to B./A.

    Those operations are of course also working for single intervals and reals, giving the results defined above. All are executed by the division functions given by the library BIAS [Rump99].

Examples

#(2,3) / 3;
9 / #(-5,6);
#(0,2) / #(3,4);
listdiv(#(-2,3),#(-1,1));

A = [1,2,3;3,4,5];
B = [2,3,4;4,5,6];
C = interval(A,B);
D = interval(-A,B);

#(0,2) \ C; // equivalent to C / #(0,2)
#(0,2) / D; // all D components include 0, the result will have only |-inf,inf| components
C / 4;

C ./ D; //elementwise division
C .\ D; // equivalent to D ./ C

E = #(ones(2,2),2) // square interval matrix with all components |1,2|
A = [1,2;3,4]  // square real matrix

E / A // equivalent to E * A^(-1)

interval square matrix or vector

n

integer

p

integer matrix or vector

s

single interval

Description

  • Since the multiplication of intervals is defined, we can then have the definition of the power of a single interval [Hansen03]. It's available here with the ^ and .^ operators.

    These operators are extended for interval matrices and vectors :

    - ^ stands for interval square matrices power (with matrix multiplication), elementwise interval vectors power and single interval power. A matrix of powers can be applied to a single interval; it will return a same size matrix with the interval at the power of each element of the power matrix.

    - .^ stands for elementwise interval matrices and vectors power (for the power of a single integer or a same size matrix/vector integer). For a single interval, it has the same use than ^.

    We notice that the negative power of an interval A^n with n < 0 will return the inverse (with the inverse of an interval defined on the division part above) of A^(-n).

    All these operations are executed by the multiplication and single interval power functions given by the library BIAS [Rump99]. For the square matrix power, it's based on a decomposition of the power on the base 2, and then calculate the several squares and combine them between each other.

Examples

#(2,3) ^ 3;

A = [1,2;3,4];
B = [2,3;4,5];
C = interval(A,B);

C .^ 2 // the square of each element is calculated
C .^ C.inf // C.inf is the matrix power
C ^ 4 // 4th matrix power
C^(-4) // inverse of the 4th matrix power
#(3,4) ^ C.inf // returns a 2*2 matrix
C(1,:)^4 // elemetwise power of the first row of C

Calling Sequence

u = v^ (1./n)
y = s^ p
y = x .^ (1./n)
y = x .^ p
y = sqrt(x)

Parameters

y, x

interval matrix or vector

r

interval vector

u, v

interval vector or single interval

n

integer

p

reals between -1 and 1 matrix or vector

s

single interval

Description

  • With the same syntaxe, uses and restrictions of the power functions .^ defined in the part just above, we can calculate the nth roots by only putting 1./n in the power parts of the expressions. It only deals with the positive part of the interval matrix/vector components. If an interval matrix/vectior component is negative, an input problem error will be returned. For the use of A.^n with A and n with the same sizes, n can have components for root and power calculations. For root calculation, ^ will only be useful for interval vectors (returning elemntwise roots) and single intervals.

    Moreover, a function sqrt has been implemented to calculate the elementwise square roots of an interval vector or matrix. All these operations are executed by the root functions given by the library BIAS [Rump99].

Examples

#(2,3) ^ (1/.3);

A = [1,2;3,4];
B = [2,3;4,5];
C = interval(A,B);

C .^ (1./2) // the square root of each element is calculated
sqrt(C) //equivalent to the line above
C .^ (1 ./C.inf) // C.inf is the root matrix 

C(1,:)^(1./5) // elemntwise 5th root

n=[2,0;1./5,1./3];

C.^n;

Description

  • An other arithmetic for intervals exists : the generalized arithmetic. This one has special mathematic properties including defining a group for the addition (meaning that X + opposite(X) =0 for X a generalized interval and with the operation opposite consisting on inverting the bounds ). Such an arithmetic is not available yet and operations with such intervals (defined with the component arith of the interval structure set as "generalized", see interval for more details) are blocked with the flag %I4S_arith.
