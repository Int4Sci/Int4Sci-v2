interval

interval: Creation of an interval vector or matrix with its inferior and superior bounds

Calling Sequence

t=interval(x,y)

Parameters

x,y

:similar size real vectors or matrices

t

interval vector or matrix

Description

  • interval(x,y) is the interval vector or matrix created with the inferior bound vector or matrix x and the superior bound vector or matrix y. The component t(i,j) of t is printed |x(i,j),y(i,j)|. Mathematicaly, it represents any real included in the interval |x(i,j),y(i,j)| [Hansen 03][Moore66].

    The notation interval(x) is equivalent to interval(x,x). Such an interval |x,x| is called a degenerate interval [Hansen 03]. If x or y are single reals, it's assumed that the lower or upper bound is constant for each t component.

    t is stored as a "mlist" named "InterM". t elements are :

    t.dims

    returns the bounds size matrix, considered as the interval size matrix.

    t.inf

    returns the inferior bound. Here t.inf is x.

    t.sup

    returns the inferior bound. Here t.sup is y.

    t.arith

    returns the arithmetic type.

    t.mode

    returns the rounding precision mode.

Examples

t=interval(1,2);
            
A=[1,2;3,4];
B=[2,3;4,5];
interval(A,B);
interval(A,5);
C=interval(A); // degenerate interval matrix

#

#:Creation of an interval vector or matrix with its inferior and superior bounds

Calling Sequence

t = #(x,y)

Parameters

x,y

similar size real vectors or matrices

t

interval vector or matrix

Description

  • The function # is a shortcut equivalent to the function interval.

Examples

t=#(1,2);

A=[1,2;3,4];
B=[2,3;4,5];
#(A,B);
C=#(A);

inf , sup

inf and sup: returns respectively the inferior and superior bound of an interval matrix or vector

Calling Sequence

x = inf(t)
x = sup(t)

Parameters

x

real vector or matrix

t

interval vector or matrix

Description

  • The function inf returns the inferior bound matrix or vector of an interval matrix or vector. In a same way, sup returns the superior bound matrix or vector.

    Warning : for direct allocations on the bounds, inf and sup can't be used. You'd better use the elements of the interval structure (see interval) .inf and .sup. For example, for an interval matrix a, a direct allocation of the inferior bound will be written with a.inf = M with M a real matrix with the same size than a.

Examples

// inf and sup use
t=#(1,2);
inf(t); //returns 1
sup(t); //returns 2

A=[1,2;3,4];
B=[2,3;4,5];
C=#(A,B);
inf(C); //returns A
sup(C); //returns B

//direct allocation on C bounds:

C.inf = [-1,-2;-3,-4] // Same C's size matrix for inferior bound
C.sup(1,1) = 20 // Change the superior bound of the first element of C
C.inf(:) = 0 // Set the inferior bound to 0

mradius

mradius:Creation of an interval vector or matrix with its middle value and its radius

Calling Sequence

t = mradius(x,e)

Parameters

x

real or interval vector/matrix

e

real vector or matrix

t

interval vector or matrix

Description

  • The function mradius creates an interval based on a middle value and a radius. Basicly, for a single real middle value x and a single real radius value e, mradius(x,e) is the interval |x-e,x+e|[Moore 66]. x can also be an interval, in which case mradius(x,e) returns the interval |x.inf-e,x.sup+e|.

    This definition is extended for this function to a real or interval middle value matrix x and a same size real radius matrix or a single real radius.

Examples

t=mradius(0,1);//returns the |-1,1| interval
t=mradius(#(-1,1),1);//returns the |-2,2| interval

A=[1,2;3,4];
B=[0.5,1;1,2];
mradius(A,B);
mradius(A,1);
C=#(B,A);
mradius(C,B);

mid

mid:returns the middle value matrix or vector of an interval matrix or vector.

Calling Sequence

x = mid(t)

Parameters

x

real vector or matrix

t

interval vector or matrix

Description

  • The function mid returns the middle value of an interval vector or matrix. For a single interval |x,y|, the middle value is mathematically defined as (y+x)/2 [Hansen 03]. For an interval matrix or vector t, this function will return a same size real matrix or vector of the middle value of each t component.

Examples

t=#(0,2);
mid(t); // returns 1

A=[1,2;-3,4];
B=[2,3;4,20];
mid(#(A,B)); // returns [1.5,2.5;0.5,12]

rad

rad:returns the radius matrix or vector of an interval matrix or vector.

Calling Sequence

x = rad(t)

Parameters

x

real vector or matrix

t

interval vector or matrix

Description

  • The function rad returns the radius value of an interval vector or matrix. For a single interval |x,y|, the radius value eis a positive real mathematically defined as e = |y-x|/2 [Hansen 03]. This definition, added with the middle value M definition (function mid) allows to write the interval |x,y| as |x,y| = |M-e,M+e| (function mradius).

    For an interval matrix or vector t, rad function will return a same size real matrix or vector of the radius of each t component.

Examples

t=#(0,2);
rad(t); // returns 1

A=[1,2;-3,4];
B=[2,3;4,20];
rad(#(A,B)); // returns [0.5,0.5;3.5,8]

wid

wid:returns the width real matrix or vector of an interval matrix or vector.

Calling Sequence

x = wid(t)

Parameters

x

real vector or matrix

t

interval vector or matrix

Description

  • The function wid returns the width value of an interval vector or matrix. For a single interval |x,y|, the width l is a positive real mathematically defined as the absolute value of the difference between the superior bound and the inferioi bound l = |y-x| [Hansen 03]. It corresponds too as the double of the interval's radius (function rad).For an interval matrix or vector t, wid function will return a same size real matrix or vector of the width of each t component.

Examples

t=#(0,2);
wid(t); // returns 2

A=[1,2;-3,4];
B=[2,3;4,20];
wid(#(A,B)); // returns [1,1;7,16]

mag

mag: returns the magnitude of an interval matrix or vector

Calling Sequence

y = mag(t)

Parameters

y

real matrix of vector

t

interval matrix or vector

Description

  • This function returns the magnitude of an interval matrix or vector. The magnitude of an interval is mathematically defined by the maximum of the absolute value of its bounds [Hansen 03]. For an interval matrix t, it returns a same size matrix with the magnitude of each component |t(i,j)|.

Examples

A=[1,2];
B=[2,3];
C=[1,-20];
D = interval(A,B); // D = [|1,2|,|2,3|]
E = interval(C,B); // E = [|1,2|,|-20,3|]

r = mag(D) //returns [2,3]
r = mag(E) //returns [2,20]

mig

mig: returns the mignitude of an interval matrix or vector

Calling Sequence

y = mig(t)

Parameters

y

real matrix of vector

t

interval matrix or vector

Description

  • This function returns the mignitude of an interval matrix or vector. The mignitude of an interval is mathematically defined by the minimum of the absolute value of any real contained in this interval [Hansen 03]. For an interval matrix t, it returns a same size matrix with the mignitude of each component |t(i,j)|.

Examples

A=[1,2];
B=[2,3];
C=[1,-20];
D = interval(A,B); // D = [|1,2|,|2,3|]
E = interval(C,B); // E = [|1,2|,|-20,3|]

r = mig(D) //returns [1,2]
r = mig(E) //returns [1,0]

Calling Sequence

t:k
t:l:k

Parameters

k

integer

t

interval matrix or vector

l

integer vector

Description

  • This functionnality allows to bissect any component of an interval matrix or vector. t:k will bissect k times all the components of t. The result is stored in a k element list.

    It's poosible to choose which t component will be bissected by executing t:l:k, where l is a t's index vector. In this case, the list returned will contain all the combinations of the k bissections for each choosen component; meaning a k^length(l) element list including the non-bissected components.

Examples

A=[1,2;-3,-4];
B=[2,3;-2,-3];
C = interval(A,B); 

D = C:1:3; // D is a 3 element list with each bissection of C(1) interval
D = C:2; // D is a 2 element list with each bissection of all C components
D = C:[1,2,4]:2 // D is a 8 element list with each bissection combination of C(1),C(2) and C(3) intervals

intersection

intersection : returns the intersection of two intervals

Calling Sequence

t = intersection(x,y)

Parameters

t,x,y

interval matrix or vector

Description

  • This function intersects two interval matrices or vectors. By definition [Moore 66], an intersection of two single intervals is the interval containing all the reals they have in common. This defintion is extended for this function to interval vectors or matices; for two same size intervals matrix or vector x and y, t = intersection(x,y) will return t such as each t's components t(i,j) is the intersection of x(i,j) and y(i,j).

    In many cases, the intersection of two intervals will return an empty set. In this toolbox's version, such an empty interval is not defined, but it is represented for this function by the interval |Nan,Nan|.

Examples

A=[1,2;-3,-4];
B=[2,3;-2,-3];
C = interval(A,B);
D = interval(-3*ones(2,2),2*ones(2,2)); // D is a 2*2 interval matrix with |-3,2|
E = interval(-ones(2,2),ones(2,2)); // E is a 2*2 interval matrix with |-1,1|

F = intersection(D,C);

F = intersection(E,C);

in

in : returns a boolean for interval inclusion

Calling Sequence

in(x,y)
G = in(x,y)

Parameters

y

interval matrix or vector

x

interval or real matrix/vector

G

boolean matrix or vector

Description

  • This function checks the inclusion of an interval matrix or vector in an other one. By definition [Moore 66], an interval X = |xi,xs| is included in the interval Y = |yi,ys| if xi > = yi and xs < = ys . If such an inclusion is verified, the scilab command in(X,Y) returns the boolean true. This defintion is extended for this function to interval vectors or matices; for two same size intervals matrix or vector x and y, t = in(x,y) will check the inclusion of each component x(i,j) in y(i,j), and return a boolean matrix or vector with x and y size.

    It's possible too to check the inclusion of a real in an interval with the same function (meaning check if the real x verifies yi < = x and ys > = x for an inclusion in the interval Y = |yi,ys|). Indeed, the first in input can be a real or an interval matrix/vector.

    Finally, an other function has been implemented to check strict inclusion (meaning the interval included has no common bound with the interval it shall be included in). This function is Strictin and has the same functionnalities than in (inclusions of reals and intervals matrix or vectors returning a boolean matrix/vector).

Examples

in(#(0,1),#(-1,1)) // returns true
in(0,#(-1,1)) // returns true

A=[1,2;-3,-4];
B=[2,3;-2,-3];
C = interval(A,B);
in(0*ones(2,2),C) // returns a 2*2 boolean matrix with false
in(A,C) // returns a 2*2 boolean matrix with true
Strictin(A,C) //returns a 2*2 boolean matrix with false
