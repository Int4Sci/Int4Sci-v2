interval

Creation of an interval vector or matrix

Calling Sequence

t=interval(x,y)

Parameters

x,y

:similar size real vectors or matrices

t

interval vector or matrix

Description

  • interval(x,y) is the interval vector or matrix created with the inferior bound vector or matrix x and the superior bound vector or matrix y. The component t(i,j) of t is printed |x(i,j),y(i,j)|. Mathematicaly, it represents any real included in the interval |x(i,j),y(i,j)| [Hansen 03].

    The notation interval(x) is equivalent to interval(x,x). Such an interval |x,x| is called a degenrate interval [Hansen 03]. If x or y are single reals, it's assumed that the lower or upper bound is constant for each t component.

    t is stored as a "mlist" named "InterM". t elements are :

    t.dims

    returns the bounds size matrix, considered as the interval size matrix.

    t.inf

    returns the inferior bound. Here t.inf is x.

    t.sup

    returns the inferior bound. Here t.sup is y.

    t.arith

    returns the arithmetic type.

    t.mode

    returns the rounding precision mode.

Examples

t=interval(1,2);
            
A=[1,2;3,4];
B=[2,3;4,5];
interval(A,B);
interval(A,5);
C=interval(A); // degenerate interval matrix
