size

size: returns the size of an interval matrix or vector

Calling Sequence

y = size(t [,sel])

Parameters

y

1x2 integer vector or integer number

t

interval matrix or vector

sel

a scalar or a character string

Description

  • This function is an overloading for intervals of the classical size scilab function. It returns by default an 1x2 integer vector with the numbers of row and columns of the interval matrix or vector t. The interval analysis theory assumes that the size of an interval is the size of its bounds [Moore 66]. size is also used with the sel string to specify which dimension to get :

    1 or 'r': to get the number of rows

    2 or 'c': to get the number of columns

    '*': to get the product of rows and column numbers

Examples

A=[1,2,3;3,4,5];
B=[2,3,4;4,5,6];
C = interval(A,B);

r = size(C) //returns the vector [2,3]
r = size(C,1) //returns 2
r = size(C,'*') //returns 6

size(#(2,3)) //returns the vector [1,1]

diag

diag: returns the diagonal of an interval matrix or creates a diagonal interval matrix based on a interval vector

Calling Sequence

[y]=diag(t,[k])

Parameters

y

interval matrix or vector

t

interval matrix or vector

k

integer (default value 0)

Description

  • This function is an overload for interval matrices or vectors of the classical diag scilab function. If the input t is an interval vector, the ouptut y returned is an interval matrix with its kth diagonal equal to t and the other components set to the degenerate interval |0,0|. By convention, k = 0 is for the main diagonal (default value), k>0 is for upper diagonals and k<0 for lower diagonals.In the other hand, if the input t is an interval matrix, the output y is a column vector storing the kth diagonal of t.

Examples

A=[1,2,3;-3,-4,-5];
B=[2,3,4;-2,-3,-4];
C = interval(A,B); // C is a 2*3 matrix

D = diag(C); //D is [|1,2|;|-4,-3|]
D = diag(C,1); // D is [|2,3|;|-5,-4|]
D = diag(C,2); // D is [|3,4|]
D = diag(C,-1); // D is [-3,-2|]

C = interval([1,2],[2,3]); //C is a row vector
D = diag(C); // D is a 2*2 interval matrix
D = diag(C,2); // D is 4*4 interval matrix

sum

sum: returns the sum of the components of an interval matrix or vector

Calling Sequence

y = sum(t)
y = sum(t,'r') or y = sum(t,1)
y = sum(t,'c') or y = sum(t,2)
y = sum(t,'m')

Parameters

y

single interval or interval vector

t

interval matrix or vector

Description

  • This function is an overload for interval matrices or vectors of the classical sum scilab function. For an interval matrix or vector t, the default action of y = sum(t) is to return the sum of all t's components in the interval y (cf addition of intervals in Int4Sci_Arith).

    y=sum(t,'r') (or, equivalently, y=sum(t,1) ) is the rowwise sum: y(j)= sum(t(:,j)) . y is a row vector.

    y=sum(t,'c')(or, equivalently, y=sum(t,2)) is the columnwise sum. It returns in each entry of the column vector y the sum : y(i)= sum(t(i,:)) )).

    y=sum(t,'m') is the sum along the first non singleton dimension of t.

Examples

A=[1,2,3;-3,-4,-5];
B=[2,3,4;-2,-3,-4];
C = interval(A,B); // C is a 2*3 matrix

D = sum(C); // D is |-6,0|
D = sum(C,1); // D is column vector
D = sum(C,2); //D is a row vector

norm

norm: returns the norm of an interval matrix or vector

Calling Sequence

[y]=norm(t [,flag])

Parameters

y

single interval or real value

t

interval matrix or vector

flag

string or integer

Description

  • This function is an overload for interval matrices or vectors of the classical norm scilab function.

    For an interval matrix t, there are two different norms defined [Hansen 03]:

    The infinite norm : a real defined as the largest row magnitude sum ( max(sum(mag(t),'c')) ). It's obtained by the default use of the norm function or setting flag as 'inf' or %inf : norm(t), norm(t,'inf') or norm(t,%inf).

    The 1_norm : a real defined as the largest column magnitude sum ( max(sum(mag(t),'r')) ). It's obtained with flag equal to 1 : norm(t,1).

    For an interval vector t, the infinite norm exists too (obtained with the same call than for interval matrices : norm(t,'inf') or norm(t,%inf) ) and is the largest magnitude of t's components ( max(mag(t)) )[Hansen 03]. The 1_norm is totally different : it's generalized to the p_norm (called by norm(t,p) with p an integer) and it's defined as the interval y such as:

    y=sum(t(i)^p))^(1/p) (cf power of intervals in Int4Sci_Arith). The default norm returned (called by norm(t) ) is for p = 2.

Examples

A=[1,2,3;-3,-4,-5];
B=[2,3,4;-2,-3,-4];
C = interval(A,B); // C is a 2*3 matrix

D = norm(C); // infinite norm
//equivalent to :
D = norm(C,'inf);

D = norm(C,1); // 1_norm

C= interval([1,2,3],[2,3,4]);

D = norm(C); // 2_norm
//equivalent to
D = norm(C,2)

D = norm(C,%inf) // infinite norm
D = norm(C,56) //56_norm, which is mathematicaly useless..

Calling Sequence

//For extraction from t or v :
a = t(i,j)
a = v(i)
b = t(i:j,k)
b = t(i,j:k)
c = t(i:j,k:l)

Parameters

i,j,k,l

integers

a

single interval

b

interval vector

t,c

interval matrices

v

interval matrix or vector

Calling Sequence

//For insertion in x or s:
x(i,j) = d
s(i) = d
x(i:j,k) = e or x(i,j:k) = e or x(i:j,k:l) = f

Parameters

i,j,k,l

integers

d

single interval or real

e

interval or real vector

x,f

interval or real matrices

s

interval or real matrix/vector

Description

  • These functionnalities are overloads for interval matrices or vectors of classical insertion and extraction scilab functions. They allow to extract/insert intervals submatrices or subvectors from/in an interval matrix or vector. It's possible too to insert a real matrix/vector in an interval matrix/vector, and vice-versa; in all the cases, it will always return a interval matrix or vector.

    We can notice that the insertion of an empty submatrix [] at the indexes [i1,..,in],[j1,...,jm] will delete the entries with such indexes and return a column vector with the entries left. Exceptions are for a row vector (that will still remain a row vector) and a single interval that will become the |0,0| degenerate interval.

    An error is returned when the submatrix to insert does not have a conform size (cf examples just below).

Examples

A=[1,2;-3,-4];
B=[2,3;-2,-3];
C = interval(A,B); 

D = C(1,1); // D is |1,2|
D = C(1,:); // D is the C first row.

C(2,2) = 3; // The (2,2) component of C is now the degenerate interval |3,3|
C(1,1:2) = #([0,5],[0,10]); // The first row of C is now the [|0,5|,|0,10|] row vector
C(2,2) = #([1,2],[2,3]); // An error will be returned, the size is not conform

C=[1,2,3];
C(1) = #(1,3); //C is the interval row vector [|1,3|,|2,2|,|3,3|]

matrix

matrix : reshape an interval vector or matrix to a different size interval matrix

Calling Sequence

y=matrix(v,n,m)
y=matrix(v,[sizes])

Parameters

v

an interval vector or matrix

n,m

integers

sizes

vector of integers

y

an interval vector or matrix

Description

  • This function has the same use than the classical Scilab function matrix :

    For an interval vector or a matrix with n x m entries, the command y=matrix(v,n,m) or similarily y=matrix(v,[n,m]) transforms the v interval vector (or matrix) into an n x m interval matrix by stacking columnwise the entries of v .

    If one of the dimension m or n is equal to -1 it is automatically assigned to the quotient of size(v,'*') by the other dimension.

Examples

A=[1,2;-3,-4];
B=[2,3;-2,-3];
C = interval(A,B); 

D = matrix(C,1,4); // D is a row vector
E = matrix(D,[2,2]); // E is C

Calling Sequence

x > y
x > = y
x < y
x < = y

Parameters

x,y

interval vectors or matrices

Description

  • An order for intervals has been implemented with these functionnalities. There are many ways to define an order for intervals [Hansen03] , [Moore66]. Here, we have choosen this definition for two intervals x = |xi,xs| and y = |yi,ys| :

    - x > y if xi > ys.

    - x >= y if xi >= ys.

    - x < y if xs < yi.

    - x <= y if xs <= yi.

    Here again, this definition is extended to interval matrix and vectors, where each component with the same indexes will be compared. A boolean matrix or vector (with the same size than the inputs) is returned.

    We can note that the comparison for orders defined above are different than comparison for equality of two intervals (== or ~=). Indeed, == will check the equality of the six components of the interval structure (dimensions, inferior and superior bounds matrix or vector and modes, see interval for more details). It returns then a boolean six elements row vector.

Examples

A=[1,2;-3,-4];
B=[2,3;-2,-3];
C = interval(A,B); 

C(1,1) >= C(2,1); // returns the true boolean
C(1,1) >= C(1,2); // returns the true boolean
C(1,1) > C(1,2); //  returns the false boolean

C > #(0*ones(2,2)); // returns a 2*2 boolean matrix with false
